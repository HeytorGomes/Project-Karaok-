### 🎨 Sistema de Design CSS

O projeto utiliza uma abordagem modular para estilos:

- **global.css** - Reset e variáveis globais
- **utility.css** - Classes utilitárias (flex, grid, padding, etc)
- **Componentes específicos** - Cada seção tem seu próprio arquivo

**Classes Utilitárias Principais:**
```css
.flex { display: flex; }
.grid { display: grid; }
.gap-* { gap: var(--gap-size); }
.px-*, .py-* { padding horizontal/vertical }
.items-center { align-items: center; }
.even-columns { grid-auto-flow: column; }
```
