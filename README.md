## Sobre o Projeto

**Zingen** é uma landing page inovadora que apresenta um aplicativo revolucionário de karaokê potenciado por Inteligência Artificial. O projeto foi desenvolvido com foco em **responsividade**, **acessibilidade** e **experiência do usuário** para promover um aplicativo que permite aos usuários cantar suas músicas favoritas com tecnologia de ponta.

### 🎯 Objetivo

O objetivo principal é fornecer uma plataforma visualmente atrativa e funcional que comunique claramente as funcionalidades, benefícios e planos de preço do aplicativo Zingen, facilitando a conversão de visitantes em usuários.

### 🌟 Diferencial

Ao contrário de aplicativos tradicionais de karaokê, o Zingen utiliza algoritmos avançados de Inteligência Artificial para:

- 🎵 **Remover vozes originais** de qualquer música automaticamente
- 📊 **Avaliar performance** com precisão em tempo real
- 🎬 **Sincronização de letras** em tempo real
- 👥 **Construcción de comunidade** com sistema de compartilhamento e feed social

## ✨ Recursos Principais

### 1. 🪄 Maior Biblioteca de Músicas de Karaokê

> "A IA remove a voz de qualquer música e a deixa pronta pra cantar, num passe de mágica!"

Com nosso algoritmo de remoção de voz baseado em IA, você tem acesso a um catálogo praticamente ilimitado de músicas prontas para karaokê. Não se limita apenas a músicas pré-gravadas - qualquer faixa de áudio pode ser processada e preparada para sua performance.

**Benefícios:**
- Acesso ao maior repertório musical
- Atualização constante de novas músicas
- Qualidade de áudio profissional
- Compatibilidade com diversos gêneros musicais

### 2. 🎮 Experiência Gamificada

Transformamos o karaokê em uma competição envolvente onde sua performance é pontuada através de um algoritmo ultrapreciso que avalia:

- ✅ Afinação vocal
- ✅ Timing e ritmo
- ✅ Consistência tonal
- ✅ Expressão emocional

**Gamification Features:**
- Sistema de pontuação personalizado
- Desafios diários e semanais
- Badges e conquistas desbloqueáveis
- Competições com amigos e comunidade
- Ranking global de talentos

### 3. 🎥 Grave suas Performances

Capture e compartilhe seus melhores momentos:

- Gravação de áudio de alta qualidade
- Gravação de vídeo sincronizado
- Editor de vídeo integrado
- Histórico de performance
- Análise de progresso ao longo do tempo

### 4. 👥 Compartilhe com a Comunidade

Faça parte de uma comunidade vibrante de apaixonados por música:

- 📱 Feed social integrado
- 💬 Comentários e reações
- 🔄 Compartilhamento de performances
- 🌍 Descoberta de talentos locais
- ❤️ Avaliação de colegas

### 5. 📝 Letras em Tempo-Real

Nunca perca o ritmo com tecnologia de sincronização avançada:

- Letras sincronizadas pixel-perfeito com a música
- Opção de exibir a voz original como guia
- Múltiplos idiomas suportados
- Ajuste automático de velocidade de leitura
- Modo profissional com notações musicais

### 🎨 Sistema de Design CSS

O projeto utiliza uma abordagem modular para estilos:

- **global.css** - Reset e variáveis globais
- **utility.css** - Classes utilitárias (flex, grid, padding, etc)
- **Componentes específicos** - Cada seção tem seu próprio arquivo

**Classes Utilitárias Principais:**
```css
.flex { display: flex; }
.grid { display: grid; }
.gap-* { gap: var(--gap-size); }
.px-*, .py-* { padding horizontal/vertical }
.items-center { align-items: center; }
.even-columns { grid-auto-flow: column; }
```
